# Employee Management Dashboard

## üìå Project Overview

The **Employee Management Dashboard** is a modern, responsive web application built using React that allows users to manage employee information efficiently.

The application includes:
- A login flow with mock authentication
- A dashboard showing employee statistics
- Full CRUD operations for employee management
- Search and combined filtering
- Profile image upload with preview
- Print-friendly employee list
- Persistent data storage using localStorage

The primary focus of this project is **clean architecture, predictable state management, and a polished user experience**, following real-world internal dashboard patterns.
